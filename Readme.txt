## OTP-Based Login
This assignment implements 2-step verification to login system by sending One Time Passowrd (OTP) once the user has been verified as registered user in the first step.

## Frontend
HTML, CSS, Javascript and FETCH API for server side communication

## Backend
1. PHP
2. PHPMailer for sending OTP to the user 

## Mysql DB
one user table with two fields email and otp.

Steps for reviewer:
Reviewer need to create two folders, 1 for front end files (index.html, dashboard.html, style.css) 
and script.js) and another for backend files(login.php, db.php,composer.lock and composer).
Also a database is to be created for the project including 2 columns,
named email and otp(with null value at the begining of the program).
